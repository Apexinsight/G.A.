# Website Image Guide

Replace any image by keeping the same filename in this folder.

- `hero-bridge.jpg` - homepage hero and mobile hero background
- `about-engineers.jpg` - homepage about section image
- `cta-engineers.jpg` - call-to-action panel and page-header background
- `project-highway.jpg` - highway project card
- `project-bridge.jpg` - bridge project card
- `project-office.jpg` - office project card
- `project-water-treatment.jpg` - water treatment project card
- `project-road-construction.jpg` - road construction project card
- `project-residential.jpg` - residential project card
- `project-urban-roads.jpg` - urban roads project card
- `project-wastewater.jpg` - wastewater project card
- `team-director-1.jpg` to `team-director-3.jpg` - leadership profile photos
- `team-engineer-1.jpg` to `team-engineer-8.jpg` - team gallery photos
- `favicon.svg` - browser tab icon using the GA logo mark

Use JPG or PNG images with the same filename. For best results:

- Hero images: around 1900px wide
- Project images: around 900px wide
- Team images: square or portrait, around 500-600px wide
